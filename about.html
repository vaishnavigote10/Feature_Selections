About This Project

Comprehensive Feature Selection & Machine Learning Analysis

üéØ Project Overview

This comprehensive feature selection project demonstrates advanced machine learning techniques applied to the Wine Quality dataset. Our goal is to identify the most critical features that impact wine quality prediction while optimizing costs and computational efficiency.

By combining five different feature selection methods and using consensus voting, we've identified 5 "Super Features" that maintain 98.4% of baseline model accuracy while reducing testing costs by 64.2% and improving model interpretability.

üìä Analysis Methodology

Five Feature Selection Techniques:

1. Correlation Analysis

Calculates Pearson correlation between each feature and the target variable, selecting features with |correlation| > 0.3.

2. F-Regression Scores

Statistical univariate testing that measures linear dependency between features and target. Selected top 8 features by F-score.

3. Tree-Based Importance

Random Forest and Gradient Boosting models compute feature importance based on how much they reduce impurity during splits.

4. Recursive Feature Elimination (RFE)

Iteratively trains models and removes features with lowest coefficients, identifying top 8 features.

5. Mutual Information & PCA

Information-theoretic approach measuring mutual dependency and dimensionality reduction through Principal Component Analysis.
